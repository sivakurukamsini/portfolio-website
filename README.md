# Kamsini Sivakuru - Portfolio Website

A modern, responsive, and animated portfolio website for software engineer internship applications.

 🎨 Features

- **Modern Design**: Deep navy background (#0a192f) with neon cyan accents (#64ffda)
- **3D Animations**: Rotating 3D cube in hero section and 3D hover effects on cards
- **Fully Responsive**: Works seamlessly on mobile, tablet, and desktop devices
- **Smooth Animations**: Fade-in effects on scroll, smooth scrolling navigation, hover animations
- **Interactive Elements**:
  - Download CV functionality
  - GitHub profile link (opens in new tab)
  - Project links (open in new tabs)
  - Smooth navigation between sections
  - Active nav link highlighting
  - Parallax effects
  - Scroll-to-top button

 📁 Files

- `index.html` - Main HTML structure with semantic markup
- `style.css` - Modern CSS with Flexbox, Grid, animations, and media queries
- `script.js` - Interactive features and animations
- `cv.pdf` - CV file for download (sample provided)

 🚀 Getting Started

1. **Open the website**: Simply open `index.html` in your web browser
2. **No dependencies**: This is pure HTML, CSS, and JavaScript - no frameworks needed!
3. **Works offline**: Download all files and open locally

 📝 Customization

### Update Your Information

1. **Name & Contact**: Edit the personal details in `index.html`
   - Line 35: Change "Kamsini Sivakuru" to your name
   - Line 67: Update email address
   - Line 68: Update GitHub username URL

2. **Add Your GitHub Profile**:
   - Search for `YOUR_GITHUB_USERNAME` in `index.html`
   - Replace all instances with your actual GitHub username
   - The link will open in a new tab automatically

3. **Update Your CV**:
   - Replace `cv.pdf` with your actual CV file
   - Make sure the filename is exactly `cv.pdf` in the same directory as `index.html`

4. **Modify Projects**:
   - Update project titles, descriptions, and links in the Projects section
   - Change the project URLs to your actual project links

5. **Customize Skills**:
   - Edit the skills list in the Skills section
   - Each skill card has an icon, title, and description

## 🎨 Color Theme

The website uses a professional tech color scheme:
- **Primary Background**: `#0a192f` (Deep Navy)
- **Accent Color**: `#64ffda` (Neon Cyan)
- **Text Color**: `#ccd6f6` (Light Gray)
- **Card Background**: `#112240` (Dark Blue)

These colors are defined as CSS variables in `style.css` and can be easily modified.

## 📱 Responsive Breakpoints

- **Desktop**: Full layout with side-by-side hero section
- **Tablet** (≤ 768px): Optimized grid layouts, responsive navigation
- **Mobile** (≤ 480px): Single column layouts, mobile-optimized spacing

## ⚙️ JavaScript Features

1. **Smooth Scrolling**: Smooth scroll behavior for navigation links
2. **Download CV**: Click handler for CV download functionality
3. **Scroll Animations**: Fade-in effects when elements come into view
4. **Navbar Effects**: Shadow effect and active link highlighting on scroll
5. **3D Hover Effects**: Perspective-based hover effects on skill cards
6. **Parallax Effect**: Subtle parallax scrolling on hero section
7. **Scroll-to-Top Button**: Automatically appears when scrolling down
8. **Mobile Menu Toggle**: Hamburger menu for mobile devices
9. **Cursor Follower**: Optional animated cursor follower
10. **Dark Mode Toggle**: Optional dark/light mode switching
